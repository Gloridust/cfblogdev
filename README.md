# cflsgx.top
源自@rene.wang，为学校开发使用。

## 配置

配置文件位于 `/data/config.json`

-   title:博客名称

*   root:博客根网址

*   author: 作者信息
    -   name:名称
    -   image:玉照
    -   intro:介绍
*   catagories: 目录配置

## 写作

将文章放入`/posts`目录即可. NBlog 支持 markdown 和 HTML.

### Atomic css

| 写法       | 可选参数    | 说明     |
| ---------- | ----------- | -------- |
| Bgc(color) | white green | 背景颜色 |
| P(padding) | 10px 20px   | 间距     |
